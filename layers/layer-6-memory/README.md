# Layer 6 — Memory

## Responsibility
Two types. Short-term: current session, active task, immediate context. Long-term: patterns observed over weeks and months, preferences established through repetition.

## What this layer owns
Session memory, long-term pattern storage, preference modeling.

## Scope
Persistent. Survives reboots. Private to the local machine.

## Status
Specification complete. Implementation private.
Build starts after Interwave ships its core.
