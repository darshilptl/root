# Layer 4 — Knowledge

## Responsibility
Everything Root has learned — structured, retrievable, continuously updated. Not a log. A knowledge base that grows more precise the longer Root runs.

## What this layer owns
Knowledge graph, retrieval index, structured memory storage.

## Scope
Read and write. Consumed by all reasoning and planning layers.
