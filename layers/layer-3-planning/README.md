# Layer 3 — Planning

## Responsibility
Before Root acts, it plans. Goal understood. Path mapped. Dependencies identified. Permission model checked. Nothing executes without a plan.

## What this layer owns
Task decomposition, dependency resolution, pre-execution validation.

## Scope
Planning-only. No execution happens here.

## Status
