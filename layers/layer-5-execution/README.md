# Layer 5 — Execution

## Responsibility
Actions happen here. File operations, application control, workflow automation — anything Root is permitted to do. Every action traceable back to a plan, a reason, and an explicit permission.

## What this layer owns
Action execution, permission verification, trace logging.

## Scope
Execution-only. Every call passes through Layer 8 before running.

## Status
Specification complete. Implementation private.
Build starts after Interwave ships its core.
